# TECNAL
**T**hreshold-cost  **E**fficiency  **C**onfiguration for  e**N**durance  **A**erial  **L**ift

A 16lb~ uav designed for range and endurance. Goals include: 25 mile range, 50min endurance, 4lbs payload, autonomous waypoint navigation w/ gps and other sensors. Bringing commercial capability at a pricepoint easily affordable for small businesses. A personal passion project, documentation is for [Blueprint](https://blueprint.hackclub.com).
